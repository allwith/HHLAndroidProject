# HHLAndroidProject
一个包含所有Android知识点的综合项目，里面包含封装好的网络库、缓存、图片加载等等

目前已经有的：
>>DrawerLayout菜单、
>>Toobar封装、
>手势滑动返回（仿微信）、
>万能适配器HHLQuickAdapter封装，支持多种布局
